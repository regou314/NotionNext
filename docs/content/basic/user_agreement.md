## 五、辅助服务与后台运行

1. **辅助服务**：若用户使用"辅助服务"功能，将能够使用模拟手势、返回、多任务等无障碍功能
2. **后台运行**：本APP主要在后台运行，需要后台自动启动权限以确保功能的稳定使用
3. **自启动与关联启动**：为保障本APP核心功能的正常运行，在以下合理使用场景中，本APP会在设备开机完成后自动启动，以恢复用户此前的功能设置：
   - **开机自启动**：设备开机完成后（接收 `android.intent.action.BOOT_COMPLETED` 广播），本APP会自动启动以恢复悬浮球、悬浮侧边栏等用户已开启的后台功能，确保用户在重启设备后无需手动重新开启即可继续使用
   - 上述自启动行为仅在用户已主动开启相应功能（如悬浮球、侧边栏开关等）的前提下触发，不会在用户未使用相关功能时频繁自启动
   - 本APP不会在未经用户知情或无合理使用场景的情况下，频繁自启动或关联启动第三方APP

## 十、会员服务协议

### 1. 会员服务概述

本APP提供会员服务（以下简称"会员"），用户开通会员后可享受会员专属功能与权益。会员服务是付费服务，用户需支付相应费用后方可开通。

### 2. 会员权益

开通会员后，用户可享受以下权益（具体以APP内实际展示为准）：

- **全部功能解锁**：解锁本APP全部功能，包括但不限于悬浮球、自动化、侧边栏等高级功能
- **专属皮肤与主题**：使用会员专属的在线皮肤与主题
- **持续更新权益**：会员有效期内，可免费使用后续新增的会员功能
- **优先客服支持**：享受优先响应的客户服务

### 3. 会员有效期

会员有效期为自支付成功之日起算的对应天数。不同套餐对应的有效期以APP内价格页面展示为准。会员到期后，会员权益将自动终止，用户需重新购买方可继续享受会员服务。

用户可通过APP内"会员中心"页面查看当前会员到期时间。

### 4. 费用与支付

- 会员费用以APP内价格页面展示的金额为准
- 我们可能会根据运营策略调整会员价格，调整后的价格将在APP内公示
- 已开通的会员不受价格调整影响，续费时以当时价格为准
- 支付方式为微信支付，由微信支付平台提供支付安全保障

### 5. 邀请码解锁

用户可通过输入有效邀请码的方式解锁会员权益。邀请码解锁规则如下：

- 每位用户仅限使用一次邀请码解锁
- 邀请码不可转让、不可兑换现金
- 邀请码解锁的会员有效期以APP内展示为准
- 我们有权对异常邀请码使用行为（如批量注册、刷取邀请码等）进行处理，包括但不限于取消已解锁的会员权益

### 6. 退款政策

**虚拟商品一经购买，原则上不支持退款。** 请在购买前仔细确认套餐内容与有效期。以下情况可联系客服协商处理：

- 重复支付同一套餐
- 支付成功后会员权益未正常到账
- 其他经我们确认的系统故障导致的异常扣款

### 7. 会员服务的中止与终止

发生以下情形时，我们有权中止或终止您的会员服务且不承担违约责任：

- 用户违反本协议或相关法律法规
- 用户利用会员权益从事违法违规活动
- **用户通过非法手段获取会员权益**（包括但不限于使用非官方渠道的破解工具、篡改应用数据等）
- 因不可抗力导致服务无法继续提供

会员服务终止后，已支付的费用不予退还。

### 8. 自动续费

本APP**暂不提供**自动续费服务。会员到期后如需继续使用，请手动重新购买。
